Web fonts
=========

Self-hosted web-font subsets used by the site.

- Source Serif 4 — title/display serif, used as a license-safe substitute for the Minion-like brand voice.
- IBM Plex Mono — accent mono, used as a license-safe substitute for the PP Fraktion Mono-like labels/navigation/buttons.

Both families are distributed under the SIL Open Font License 1.1 via Google Fonts. Do not replace these files with the bundled branding OTF files unless a web-embedding license is confirmed.
